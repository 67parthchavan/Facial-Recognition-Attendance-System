# 🎯 Facial Recognition Attendance System

## 📌 Overview

A Python-based application that uses computer vision to detect and recognize faces in real time and automatically mark attendance with timestamps. This system reduces manual effort and improves accuracy.

---

## ✨ Features

* Real-time face detection and recognition
* Automatic attendance marking
* Stores records with date and time
* Simple and efficient workflow

---

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy
* Pandas

---

## ⚙️ How It Works

1. Capture and store face images
2. Train the recognition model
3. Detect and recognize faces via webcam
4. Mark attendance automatically

---

## 🚀 Run the Project

```bash
pip install -r requirements.txt
python main.py
```

---

## 📂 Project Structure

```bash
dataset/        # Face images  
trainer/        # Trained model
